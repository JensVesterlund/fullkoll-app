# Behörigheter – varför vi behöver dem

- Kamera: För att skanna kvitton med OCR och läsa QR/streckkoder.
- Notiser: För att skicka påminnelser om utgångsdatum, budgetvarningar och abonnemangsdragningar.
- Filer/Media: För att importera kvitton från bilder/PDF och exportera PDF/CSV.
- Nätverk: För säker säkerhetskopia/synk och uppdatering av data mellan enheter.

Du kan när som helst ändra behörigheter i systeminställningarna.
